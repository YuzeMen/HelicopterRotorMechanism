# UH-60 Inspired Helicopter Rotor Head and Swashplate Mechanism

## Overview

This project is a mechanical CAD model of a helicopter rotor head and swashplate system inspired by the UH-60. As the first project heading into the start of my first year mechanical engineering journey at UWaterloo, my primary focus for this project is the mechanical design and kinematic behavior of the rotor system rather than aerodynamic optimization. The model explores how pilot control inputs are mechanically translated into blade pitch changes through the rotating and nonrotating swashplates, as well as pitch-link system for both the main and tail rotor systems.

The assembly was designed in Autodesk Fusion 360 with an emphasis on parametric modeling, mechanical relationships, and realistic motion behavior.

---

## Project Goals

* Develop an understanding of helicopter rotor control mechanisms
* Model a functional collective and cyclic pitch system
* Explore mechanical linkages used in aerospace systems
* Practice parametric CAD assembly design
* Create a foundation for future mechanical simulation and analysis

---

## Mechanical Features

The swashplate mechanism allows three primary control inputs:

* **Collective pitch:** Changes the pitch of all rotor blades simultaneously
* **Cyclic pitch:** Tilts the swashplate to create varying blade pitch throughout rotation
* **Yaw control** By collectiely changing the AoA of the tail rotor blades

The project includes both rotor assembly and a simplified drive system while intentionally simplifies certain components.

The rotor blades are modeled as simplified aerodynamic surfaces because the primary objective is studying mechanical motion and linkage behavior rather than detailed aerodynamic performance. Engine and hydraulic lines are shortened to the most immediate components affecting the rotor assembly
